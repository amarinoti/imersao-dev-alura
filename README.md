O projeto se trata de um site dinâmico pesquisavel, em uma base de itens livros de referência em Tecnologia. Composto por JavaScript, HTML, CSS e uma base de dados cadastrado em um arquivo JSON, que pode ser alimentada manualmente ou pedir sugestão ao Google Gemini, caso queira aumentar a base de dados de itens. Na página inicial aparecem apenas 5 itens e os demais são buscados na ferramenta de pesquisa, por nome do livro, autor, editara ou um termo; com link acessivel para compra do livro, caso desejar!
